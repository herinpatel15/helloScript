# Keyword

- `chhap` print
- `dabbu` variable
- `havaChustDabbu` constant
- `jo` if
- `joHaji` else if
- `nahito` else
- `chakedu` for
- `jyashudhi` while
- `kamBol` function
- `baraber` true
- `nahoy` false
- `ubhore` break
- `chalvade` continue
